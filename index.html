Documentação Words API

POST /vowel_count

Conta o número de vogais em cada palavra fornecida.

Requisição:

POST "http://127.0.0.1:5000/vowel_count"
Content-Type: application/json

{
    "words": ["Darth Vader", "Han Solo", "Mestre Yoda", "Obi-Wan Kenobi"]
}

Resposta:

{
    "Darth Vader": 3,
    "Han Solo": 3,
    "Mestre Yoda": 4,
    "Obi-Wan Kenobi": 6
}

POST /sort

Ordena as palavras em ordem crescente ou decrescente.

Requisição:

POST "http://127.0.0.1:5000/sort"
Content-Type: application/json

{
    "words": ["Darth Vader", "Han Solo", "Mestre Yoda", "Obi-Wan Kenobi"],
    "order": "asc"
}

Resposta:

[
    "Darth Vader",
    "Han Solo",
    "Mestre Yoda",
    "Obi-Wan Kenobi"
]
POST "http://127.0.0.1:5000/sort"
Content-Type: application/json

{
    "words": ["Darth Vader", "Han Solo", "Mestre Yoda", "Obi-Wan Kenobi"],
    "order": "desc"
}

Resposta:

[
    "Obi-Wan Kenobi",
    "Mestre Yoda",
    "Han Solo",
    "Darth Vader"
]

Erros

As respostas de erro possíveis incluem:

  • 400 Bad Request: O conteúdo da requisição está incorreto ou faltando.
  • 404 Not Found: O endpoint requisitado não existe.
  • 405 Method Not Allowed: O método HTTP usado não é permitido para este endpoint.
